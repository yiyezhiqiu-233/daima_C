## 像素武侠：影之刃 - 四关试炼
江湖浮沉 — 一款纯前端像素风横版武侠动作游戏，支持单人闯关与双人本地合作。

## 游戏简介
《像素武侠：影之刃 - 四关试炼》是一款以中国传统武侠为题材的 Canvas 像素动作游戏。玩家扮演一位侠客，在风雪、暗道、幻境与最终试炼中闯过四关，面对盾牌守卫、飞刀刺客、近战Boss等敌人，利用时间静止、时间回溯、假马替身等道具化险为夷。通关后，游戏将根据玩家在整个旅程中的战斗数据、死亡方式与行为习惯，计算独特的江湖命格，为每位侠客书写一段专属的江湖评语。

游戏全程在浏览器内运行，无需安装，单文件即可启动，支持单人模式与双人分屏合作模式。

## 游戏模式
单人模式
经典闯关体验。玩家独自面对四关挑战，操控一名侠客穿越所有关卡，最终根据个人战斗数据生成江湖命格评定（16种命格 + 1种隐藏命格）。

双人分屏模式
两名玩家共用一台设备，画面上下分屏显示，各自拥有独立的摄像机视角。两名玩家在同一关卡中协作闯关，需要互相配合通过石门、陷阱等机关。通关后生成双人命格评定（6种命格），综合评判两位玩家的默契程度。

## 关卡介绍

第一关：风雪夜战
场景：风雪纷飞的雪原旷野，全屏白色雪地覆盖
任务：击杀吭哧二将（盾牌守卫组合）
敌人：哧吭（白甲盾兵）、吭哧（黑甲高盾兵）
特点：敌人使用盾牌正面防御，需绕至背面攻击；场景中有草垛掩体可供躲避
第二关：迷途暗影
场景：圆形暗室 + 长暗道 + 狭窄甬道 + 隐藏密室
任务：击杀所有敌人，善用道具
敌人：4个巡逻刺客 + 4个快速巡逻兵
特色机制：
升降石门：通过后自动关闭，双人模式延长关门时间
时间静止宝石：拾取后可让时间暂停4秒，敌人静止不动，玩家速度大幅提升
假马道具：骑乘后可抵挡飞刀攻击
隐藏密室：甬道上方的小密室中藏有假马
第三关：幻境试炼
场景：全空间战斗的宽阔走廊，上下通道分离
任务：击杀所有敌人
敌人：5个固定射手 + 2个快速巡逻兵

## 特色机制：

时间回溯宝石：拾取后获得一次死亡回溯能力，可倒流最多5秒的时间
保镖令牌：拾取后召唤吭哧二将作为保镖，自动抵挡飞刀（每个保镖可挡5次）
替身系统：使用回溯后，玩家死前的替身将留在原位继续战斗
第四关：最终试炼
场景：风雪交加的超长雪原（1920像素宽地图），遍布掩体与减速区
任务：不要杀任何人，站到终点的绿宝石上通关
敌人：1个近战Boss（高生命值、高攻击力、自动追踪巡逻）
特殊规则：
Boss杀死玩家 → 玩家在起点软重生（不判定失败，不触发回溯）
玩家杀死Boss → Boss在3秒后自动复活（不判定失败，不触发回溯）
通关条件：单人模式 - 玩家站在绿宝石上；双人模式 - 两名玩家都站在绿宝石上
Boss的存亡与通关无关，核心目标是穿越Boss的防线到达终点
可用道具：时间静止宝石、时间回溯宝石、保镖令牌
地形：多处减速区（深雪地带）降低玩家移动速度
## 道具与机制
道具 / 机制	说明
时间静止宝石	拾取后按空格激活，让时间暂停4秒。期间所有敌人冻结，玩家速度提升至约3.3倍，可自由移动到敌人身后标记击杀点。静止结束后标记处的敌人会被瞬间斩杀
时间回溯宝石	拾取后获得一次死亡回溯能力。死亡时自动触发，时间倒流至回溯前最多5秒的状态，玩家位置、敌人状态、道具等全部恢复。使用后替身留在死亡位置继续战斗
替身假马	第二关隐藏道具，骑乘后可抵挡敌人的飞刀攻击（共享护盾）
保镖令牌	拾取后召唤吭哧二将作为保镖，始终在玩家前方移动，自动抵挡飞刀（每人5点生命值）
绿宝石存档点	分布在关卡中，触碰后自动存档。死亡后按R键从最近存档点复活
草垛掩体	第一关散布的掩体，可阻挡视线和飞刀
连杀系统	2.5秒内连续击杀3个敌人触发三连杀特效
江湖命格系统
游戏的核心特色之一。通关后根据玩家在整个游戏中的8项战斗统计数据，通过组合算法生成独特的四字命格代码，对应16种命格之一（另有1种隐藏命格）。

## 统计维度（8项数据）
数据项	说明
总死亡次数	整个游戏过程中死亡的总次数
跌落陷阱次数	被关卡陷阱击杀的次数
飞刀击杀次数	被单把飞刀击杀的次数
万箭穿心次数	被多把飞刀同时击杀的次数
落石压扁次数	被石头砸死的次数
Boss击杀次数	击杀Boss的总次数
时间回溯使用次数	使用时间回溯能力的总次数
时停击杀数量	在时间静止中击杀敌人的总数
命格构成
命格由四个维度组合而成：

轮回前缀（根据总死亡次数）：初入江湖 / 百死余生 / 千劫不灭 / 万冢之主
身法字（根据陷阱跌落次数）：惘魄 / 定魄
历战字（根据Boss击杀次数）：魁首 / 稚子
宿命字（根据回溯使用次数）：逆命 / 凡骨
杀性字（综合杀伤与被杀伤数据）：魔罗 / 仁侠

## 技术规格
项目	说明
渲染方式	HTML5 Canvas，像素级逐帧绘制
画面分辨率	单人 320×180，双人 320×360（上下分屏）
显示比例	3倍放大，单人 960×540，双人 640×720
像素风格	image-rendering: pixelated 锐利像素缩放
音效系统	Web Audio API 程序化合成（攻击、命中、时停、回溯等）
存档系统	localStorage 本地存档
字体	方正像素15（FZPixel）自定义像素字体
运行环境	现代浏览器（Chrome / Edge / Firefox），无需服务器
文件结构	单HTML文件，无外部依赖

## 操作提示

1. **标题画面**：鼠标选择「单人模式」「双人模式」「关于作者」
2. **道具提示**：右上角显示已拾取道具名称，鼠标悬停可查看功能说明
3. **任务目标**：左下角显示当前关卡任务提示
4. **第四关提示**：通关条件是站到终点绿宝石上，不必与Boss战斗；Boss击杀/被杀均不会导致失败
